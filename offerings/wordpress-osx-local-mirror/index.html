HOWTO: Set up a Local Mirror of your Wordpress Blog on MacOS X

This page was once the home for some instructions on how to set up a WordPress mirror on Mac OS X. For a while there, I stopped running WordPress and so the content was getting more and more obsolete. Hence I contributed the material to the WordPress Codex in the hope that someone will update it and that more people will find it useful. Now I’m running Wordpress again, but I’ll probably just update the Codex instead of this page.
